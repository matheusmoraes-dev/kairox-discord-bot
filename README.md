🤖 Kairox - Bot de Economia

O Kairox é um bot de economia desenvolvido para Discord, com foco em gerenciamento de moedas virtuais dentro de servidores.

O projeto foi criado com o objetivo de aprimorar conhecimentos em desenvolvimento de bots, integração com banco de dados e criação de sistemas interativos.

## 🚀 Funcionalidades
- Sistema de saldo
- Comandos de economia
- Integração com banco de dados
- Respostas automatizadas
